## 一句话结论

**新手前 3-6 个月：8-9ft Softboard，二手 ¥15,000-25,000。别买短板。**

看完这一节你会明白为什么。

## Softboard vs Hardboard

|  | Softboard（软板） | Hardboard（硬板） |
|---|---|---|
| **材质** | 表面海绵 + 内部泡沫 | 玻璃钢 + PU/EPS 芯 |
| **重量** | 轻 | 稍重 |
| **浮力** | **超大**（容易站起来） | 靠板型决定 |
| **安全** | ✅ 撞人不太伤 | ❌ 撞人可能伤 |
| **保养** | 免保养 | 要打蜡、修龙骨 |
| **价格** | ¥15,000-40,000 | ¥30,000-80,000+ |
| **进阶潜力** | 有限 | 高 |

**新手直接选 softboard**。硬板等你能 pop-up + 直行 + 会 bottom turn 之后再上。

## 长度选择

|  | 板长 | 适合 |
|---|------|------|
| **Longboard 长板** | 9ft+（274cm+） | 新手到高级，稳定，浪小也能玩 |
| **Funboard 中长板** | 7-8ft（213-243cm） | 新手到中级，比长板灵活 |
| **Softboard 特殊** | 8-9ft | **新手第一块的首选** |
| **Shortboard 短板** | 5'6"-6'6" | 中高级，追大浪 |

**新手买 8ft 或 9ft 的 softboard。** 太短浮力不够（追不上波），太长（10ft+）你搬不动、划不快。

## 日本可以买板的地方

### 新板

- **Murasaki Sports** —— 日本最大冲滑板零售连锁 —— 品种全但价格贵
- **BC Surf**、**Twelvegrafics**、**Sunhat** 等湘南本地精品店 —— 有品味、价格中等
- **Wallmart Softboard**（网购）—— 便宜的 softboard，¥25,000

### 二手（新手强推）

- **Mercari（メルカリ）** —— 日本最大二手 App，冲浪板超多。搜"サーフボード ロング"
- **Yahoo!オークション（Yahoo 拍卖）** —— 便宜、能砍价、但要等
- **各家 surf shop 的二手区** —— 你能看到实物，避免网购踩雷
- **Facebook Marketplace（Kanagawa Surfers 群）** —— 老外圈子转手，英语沟通方便

### 二手挑选要点

- ✅ **底部**：龙骨盒无裂、板底无明显凹陷
- ✅ **板缘**：无缺角、无深裂
- ✅ **蜡下面**：擦干净蜡看有没有裂缝
- ✅ **龙骨（fin）**：完好、不弯
- ✅ **脚绳插孔**：不松
- ❌ **发黄的旧板**：可能长期日晒、材料脆化
- ❌ **修补痕迹超过 3 处**：结构可能有问题

## 预算参考

| 预算 | 方案 |
|------|------|
| **¥15,000-20,000** | Mercari 二手 8ft softboard，够用 |
| **¥25,000-35,000** | 新 softboard（Wavestorm、Catch Surf 等） |
| **¥35,000-50,000** | 二手中长板（好的一手用过 1-2 年的） |
| **¥60,000-100,000** | 新中长板 or 二手定制长板 |
| **¥100,000+** | 新硬板（等你会 bottom turn 再考虑） |

## 除了板之外，你还需要买

|  | 建议价 | 备注 |
|---|--------|------|
| **脚绳（Leash）** | ¥3,000-5,000 | 长度 = 板长，别贪便宜（断了很危险） |
| **蜡（Wax）** | ¥500-1,000 | 按水温买（Cool / Warm / Tropical） |
| **湿衣（Wetsuit）** | ¥15,000-40,000 | 关东全年建议 3/2mm + 5/3mm 各一套 |
| **板袋** | ¥5,000-10,000 | 防止运输时磕碰 |
| **蜡刷** | ¥500 | 清蜡用 |

## 常见坑

- 🚨 **网上买超便宜（低于 ¥10,000）的板** → 可能材料很差、下水就漏
- 🚨 **朋友送的短板** → 短板难度高、别贪心
- 🚨 **一开始就买"梦想中的板"（硬板+短）** → 用不了，最后转手
- 🚨 **不买脚绳** → 板飞走砸人，甚至砸自己

## 我的建议购买路径

1. **月 1-3**：租板（湘南 4-500¥/次），确定自己真的想坚持
2. **月 4**：Mercari 买二手 8ft softboard（¥15,000-25,000）+ 脚绳 + 蜡
3. **月 6**：再买湿衣（观察季节）
4. **年 1 后**：如果想升级，二手 funboard 或 longboard
5. **年 2 后**：如果动作稳定，考虑第一块硬板

**别一次性买全套。** 冲浪需要坚持一段才能确认是不是长期爱好。
