## 为什么这一节比技术还重要

你可以浪追不好、板站不稳，日本人不会说什么。

**但你违反礼仪，全场会立刻沉默，然后你以后再也别想玩得开心。**

日本冲浪圈的"local 意识"极强。你是外来者，**低调、谦逊、遵守规矩**是你唯一的通行证。

## 一、Drop-in 铁律（最重要）

**Drop-in** = 别人已经起了这道浪、正在跑，你也起了同一道浪，挡在他前面。

**这是日本冲浪世界里最恶劣的行为**。会：
- 被本地人当场大喊 "**おい！**"
- 被瞪一整天
- 严重时被打
- 传遍这个浪点，你以后成"那个不懂规矩的外国人"

**怎么避免？**

1. **起波前**：环顾四周 30°扇形，看有没有人已经起了
2. **看到有人已经起了**：**立刻不追**，转身让开
3. **不确定谁先起的**：宁可放弃，也不要抢

## 二、优先权规则

同一波，谁先起？

**规则：离 peak 最近的那个人有优先权**。

```
         peak
          ↓
      ╱╲
     ╱  ╲
    A    B    C
   (优先) 
   
A 最靠近 peak → A 有优先权，B 和 C 必须让
```

- **你在 B 或 C 位置** → 看到 A 也在起，**立刻放弃这波**
- **你是 A**（少见）→ 尽情起，其他人会让

## 三、对本地人的态度

**打招呼：**
- 早上到浪点，遇到眼神接触 → **点头 + 「おはようございます」**
- 别硬要英语交流，本地人多半不擅长
- 会日语说日语，不会就微笑点头

**在水里：**
- 别霸占 peak（新手别坐 A 位置，坐 B 或更外围）
- 别一直问问题，观察就好
- 让本地长者、老手先追波

**离开：**
- 「お先に」（我先走了）或点头再走

## 四、停车与场地

**停车：**
- 用**正规停车场**（湘南沿线都有，¥500-1000 一天）
- **别停在本地民居前**、别停在私人门口
- **别锁自行车/摩托在别人门口**

**沙滩：**
- 换衣服**在停车场或洗手间**（不要在沙滩上暴露）
- 蜡屑、水瓶、烟头**带走**（日本没有海滩垃圾桶）
- 装备**别占地方**（长板别横放挡路）

## 五、拍照禁忌

**不能拍：**
- ❌ **本地人的脸**（未经许可）
- ❌ **停车场的车牌**
- ❌ **民居**
- ❌ **本地商店内部**

**可以拍：**
- ✅ 你自己 + 你朋友
- ✅ 海景空镜（背景没识别人的）
- ✅ 你的板、装备

**发社交媒体：**
- 别定位到**具体浪点名**（暴露了会招人）
- 别标注**本地人**（除非他们同意）
- **模糊背景的人脸**

## 六、中国游客常犯的雷（要避免）

- 🚨 **大声用中文喊朋友** → 全场侧目
- 🚨 **不看别人直接冲波** → drop-in 直接来一次
- 🚨 **穿着湿衣衣进本地便利店** → 老板脸黑
- 🚨 **占用免费淋浴太久**（超过 3 分钟）
- 🚨 **群体行动 + 高声聊天** → 破坏本地的安静氛围
- 🚨 **在浪点拉横幅、直播卖货**（真有人这么干）

## 七、"融入本地"的加分动作

- 📌 **认识 1-2 个本地人**（去装备店多聊两句、去咖啡店常来）
- 📌 **参加清沙滩活动**（Surfrider Japan 每月组织）
- 📌 **学几句冲浪日语**：「今日いい波ですね」「ありがとうございました」
- 📌 **不 drop-in、不占 peak** —— 3 个月后本地人会开始跟你点头
- 📌 **记住浪点节奏** —— 老手能预判浪、你也慢慢能，本地就认你

## 一句话总结

> **像客人一样低调、像本地人一样懂规矩。剩下的技术慢慢练。**
